# Scrabble 
- use main.py to run 
- use test_game.py to test game logic and test_cli.py to test cli logic

## Rules
- Place letters on the board to form words
- Each letter is worth one point
- For the command line use the syntax `$ word row col direction `
- Ex) `$ Hello 3 5 left`

## Requirements
- Python 3.6^
